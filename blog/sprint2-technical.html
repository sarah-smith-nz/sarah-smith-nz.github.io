Understanding ID and Class Names

What are they?

We can use CSS to make certain parts of a web page look different to others. We use selectors to 'select' the part we want to change the look of. Parts are called elements. ID and Class are both types of selectors and are used to select certain elements within a website. This allows the certain look or style to be applied to only those elements

How are they used?

ID can only be used on one type of element within a web page. Elements like head, foot, or body that will only be used once can be a good place for id selectors. First, add the ID tag in your HTML document to the element you want to style. In this example we will style the head element.



Then define this tag in the CSS stylesheet. The ID tag can be defined with a hashtag




Class selectors however can be used with multiple elements so the style can be applied to more than one type of element. Class is best used for elements like paragraph, link, headers, button or containers like div. To use Class, we first edit the HTML document to include the Class tag. This example edits a header element.



Then we edit the CSS document to assign the element a design.



When to use them

ID has it's limitations and can be useful when you want to link somewhere on your page, as there will only be one spot where that id selector is. This can be helpful whe using JavaScript. Class on the other hand is great for when you want to apply a style to multiple elements. Fun fact: Class selectors seem to be used more often than ID selectors and you can use both ID and Class together.
